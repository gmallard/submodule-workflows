Background

This project was started after following the submodule totorial at git.or.cz, and subsequently feeling the need for more detailed submodule workflow examples and explanations.

To that end the examples herein were developed on a private system, and made public on April 13, 2008, hosted at GitHub. To clone that public repository use:

  git clone git://github.com/gmallard/submodule-workflows.git

Before You Start

Before attempting to run any examples, you might want to modify the ./common/setvars script and change the user= shell variable to an appropriate user ID (if your ID can not be used).

You should also review the next section covering Administrative Requirements.
