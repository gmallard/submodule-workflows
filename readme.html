git Submodules Tests and Examples


Background

This repository was started after following the submodule totorial at git.or.cz, and subsequently feeling the need for more submodule examples and explanations.

To that end the examples herein were developed on a private system, and made public on April 13, 2008, hosted at GitHub. To clone that public repository use:

git clone git://github.com/gmallard/submodule-workflows.git
