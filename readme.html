Background

This repository was started after I followed the submodule totorial at git.or.cz, and I felt the need for more submodule examples and explanations.

To that end the examples herein were developed on a private system, and made public on April 13, 2008, hosted at GitHub. To clone that repository use:

git clone git://github.com/gmallard/submods-tests.git
