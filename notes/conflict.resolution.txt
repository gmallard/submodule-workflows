# If you get merge conflicts when you 'pull and push', and they are normal file conflicts, fix 
# them per normal procedures.  This will look something like:

git pull
remote: Counting objects: 5, done.
remote: Compressing objects: 100% (2/2)remote: , done.
remote: Total 3 (delta 0), reused 0 (delta 0)
Unpacking objects: 100% (3/3), done.
From /home/gallard/subtut/public/a/
   87ff7c1..7eb223f  master     -> origin/master
Auto-merged a.txt
CONFLICT (content): Merge conflict in a.txt
Automatic merge failed; fix conflicts and then commit the result.

# Fix per normal procedure
vi a.txt
git add a.txt
