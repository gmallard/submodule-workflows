The Empty Supermodule Examples


Introduction

This page describes the empty supermodule examples.

System Requirements

Required components and environment are:

  • An installed and working version of git
  • A top level direcory named /public/repos that is writable by anyone. To change this name, you must modify the scripts.

Instructions

Change to the empty-super subdirectory, and run the scripts in the following order:

  1. ./gen.shared.submods.sh
  2. ./gen.shared.empty.super.sh
  3. ./set-perms.sh
  4. ./add.submod.sh
  5. ........../uc-001.sh

The latter script may be executed multiple times from either of the repo direcotries created by add.submod.sh.

Other users may participate by:

  1. ........../get.submods.sh
  2. ........../uc-001.sh
