The Empty Supermodule Examples


Introduction

This page describes the empty supermodule examples.

System Requirements

Required components and environment are:

  • An installed and working version of git
  • A top level direcory named /public that is writable by anyone. To change this name, you must modify the scripts

Instructions

Change to the tutorial subdirectory, and run the scripts in the following order:

  1. ./gen.shared.submods.sh
  2. ./gen.shared.empty.super.sh
  3. ./add.submod.sh
  4. ./get.submods.sh
  5. ./uc-001.sh

The latter script may be executed multiple times.
