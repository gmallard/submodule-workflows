The Salted Supermodule Examples


Introduction

This page describes the salted supermodule examples.

System Requirements

Required components and environment are:

  • An installed and working version of git
  • A top level directory named /public that is writable by anyone. To change this name, you must modify the scripts

Instructions

Change to the tutorial subdirectory, and run the scripts in the following order:

  1. ./gen.shared.sh
  2. ./uc-001.sh

The latter script may be executed multiple times.
