The Salted Supermodule Examples


Introduction

This page describes the salted supermodule examples.

System Requirements

Required components and environment are:

  • An installed and working version of git
  • A top level directory named /public/repos that is writable by anyone in the 'git' group. To change this name, you must modify the scripts

Instructions

Change to the salted-super subdirectory, and run the scripts in the following order:

  1. ./gen.shared.sh

From different users, or multiple instances of the same user, arrange to:

  1. Create a private work directory
  2. CD in to that directory
  3. Have the scripts in $PATH

And then in each user or instance, do:

  1. clone-001.sh

And then:

  1. uc-001.sh

The latter script may be executed multiple times.
