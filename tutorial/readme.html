Introduction

This page describes the scripts contained in the tutorial subdirectory of this repository.

The scripts are based directly from examples available online. These include:

  1. Git Submodule Tutorial
  2. The Git User Manual

System Requirements

Required components and environment are:

  • An installed and working version of git

Nothing else, really.

Instructions

Change to the tutorial subdirectory, and run these scripts in the following order:

  1. ./tut-flow.sh
  2. ./uc-001.sh

The latter script may be executed multiple times.

Instructions With Bare Public Repos

This is a modified tutorial scenario, where the public repositories used are created via clone with the --bare option.

Change to the tutorial subdirectory, and run these scripts in the following order:

  1. ./tut-flow-bare.sh
  2. ./uc-001.sh

The latter script may be executed multiple times.
